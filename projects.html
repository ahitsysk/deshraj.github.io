Gradient-weighted Class Activation Mapping(Grad-CAM) Demo

Gradient-weighted Class Activation Mapping (Grad-CAM) is a novel class-discriminative localization technique, that can be used to make CNN based models interpretable. Grad-CAM highlights regions of the image the VQA model looks at while making predictions. Given an image and a caption or question about that image, the model shows where it looked while doing prediction.

Indoor Navigation Android Applocation(Using Beacons)

Beacons and Stickers are small wireless sensors that you can attach to any location or object. They broadcast tiny radio signals which your smartphone can receive and interpret, unlocking micro-location and contextual awareness. The navigation for the different Office buildings and malls can be made much simpler using Beacons. The user needs to install the application and he can navigate easily to the nearby locations like different stores in a Mall. Implemented the whole android application along with the use of A* Search Algorithm for finding the shortest route

Public Health Surveillance System
